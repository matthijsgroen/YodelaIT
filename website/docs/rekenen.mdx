# Rekenen

Levenslied ondersteunt de rekenkundige operatoren `+`, `-`, `*` en `/`. De taal bevat aliassen voor elke operator, zodat u tekstueel aangename uitdrukkingen kunt schrijven.

## Aliassen voor rekenkundige bewerkingen

- optellen `met`
- aftrekken `zonder`
- vermenigvuldiging `keer`, `van`
- deling `verdeelt over`, `boven`

Voorbeelden:

- <LineExample>Stop de passie van jouw hart in mijn handen</LineExample> - vermenigvuldig
  `jouw hart` met `de passie` en zet het resultaat in `mijn handen`
- <LineExample>Laat mijn wereld niets zonder jouw liefde wezen</LineExample> - Initialiseer
  `mijn wereld` met het resultaat van 0 - `jouw liefde`
- <LineExample>Als de tranen van een kind bestaat uit niets</LineExample> - controleer
  of `de tranen` \* `een kind` = 0
- <LineExample>{`laat mijn gevoelens mijn hoofd boven de wolken wezen`}</LineExample> -
  Deel `mijn hoofd` door `de wolken` en zet het resultaat in `mijn gevoelens`

## Afronden

<Example>{`
X is 1.2
versterk X
schreeuw X
(toont 2)
`}</Example>

<Example>{`
X is 1.2
demp X
schreeuw X
(toont 1)
`}</Example>

<Example>{`
De radio is aan. Er klinkt muziek
(initialiseer de radio met 3.266)

Zeg de radio
(toont 4)
`}</Example>
